# Vague UI

A modern Vue 3 component library focused on clean design, customization, and developer experience.

## Installation

Install components using the Vague CLI:

```bash
vague add button
vague add card
```

## Available Components

### UI
- Button
- Card
- Input
- Select

### Layout
- Container
- Spacer

### Feedback
- Form

## Development

Components are distributed through the Vague registry and can be installed individually using the CLI.
